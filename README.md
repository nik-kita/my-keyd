# [Nix determinate](https://determinate.systems/nix-installer)
* install
  ```
  curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install --determinate
  ```
* uninstall
  ```
  /nix/nix-installer uninstall
  ```
